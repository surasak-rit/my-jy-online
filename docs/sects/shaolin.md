# 🙏 วัดเส้าหลิน (少林) — เอกสารออกแบบสำนัก

> "天下武功出少林 — วิชายุทธ์ใต้หล้า ล้วนกำเนิดจากเส้าหลิน"

สำนักต้นธารแห่งวิทยายุทธ์ ตั้งบนเขาซงซาน เป็นอารามพุทธที่สืบวิชาจากพระโพธิธรรม (達摩 / ตักม้อ)
รับเฉพาะ **ศิษย์ชาย ♂** ตามกฎอารามแต่โบราณ (อิง GDD §門派 — เส้าหลินรับเฉพาะตัวละครชาย)

---

## 1. เนื้อเรื่อง (เส้นเรื่องหลัก)

เส้าหลินคือ "บ้านหลังแรก" ของผู้ฝึกยุทธ์ ใครเข้าวัดต้องเริ่มจากศูนย์ — กวาดลาน แบกน้ำ
ตีหมัดอรหันต์วันละพันครั้ง จนรากฐานมั่นจึงค่อยขึ้นสายวิชาสูง

ปมเรื่อง: ในอดีตมี **หัวหน้าครัวไฟ (火工頭陀)** แอบขโมยฝึกวิชาในยามวิกาล ไร้อาจารย์กำกับปราณ
สุดท้ายวิชาเข้าสู่ทางมาร ก่อเหตุนองเลือดในวัดแล้วลัก **คัมภีร์เก้าสุริยัน (九陽神功)** หนีไป
ผู้เล่นที่ฝึกถึงขั้นสูงจะได้รับมอบหมายจาก **หลวงจีนกวาดลานไร้นาม (掃地僧)** ผู้เป็นปรมาจารย์เร้นกาย
ให้ตามทวงคัมภีร์คืน — ปิดฉากด้วยการได้สืบวิชาภายในขั้นสุดยอดของสำนัก

สาส์นของเรื่อง (สะท้อนในบทพูด NPC): **"วิชาแรงต้องมีปราณในกำกับ มิเช่นนั้นพาตนเข้าสู่มาร"**
— เป็นแกนความสัมพันธ์ของระบบวิชาทั้งสำนัก (ดูข้อ 4)

## 4. ความสัมพันธ์ของวิชาทั้งสำนัก

### แผนผังสาย (skill tree เชิงออกแบบ)

```
                      ┌──────────────── 外功 (ภายนอก) ───────────────┐
                      │  羅漢拳 หมัดอรหันต์  (รากฐาน, ทุกคนเริ่ม)        │
                      │        │                                      │
                      │        ├──► 韋陀掌 ฝ่ามือเวยถัว (จู่โจมหนัก)     │
                      │        └──► 達摩棍 พลองปราบมาร (กวาดวงกว้าง)    │
                      └───────────────────────────────────────────────┘
                                       ▲ ปราณหล่อเลี้ยง │
                      ┌──────────────── 內功 (ภายใน) ────┴────────────┐
                      │  金鐘罩 ระฆังทอง (เกราะปราณ — อยู่รอด)          │
                      │        │                                      │
                      │        └──► 易筋經 เปลี่ยนเส้นเอ็น (แกนกลาง)    │
                      │                    │  ปลดล็อกด้วยเควสต์ลับ      │
                      │                    └──► 九陽神功 เก้าสุริยัน    │
                      └───────────────────────────────────────────────┘
                      ┌──────────────── 輕功 (ตัวเบา) ────────────────┐
                      │  一葦渡江 ข้ามชลด้วยอ้อเดี่ยว (เคลื่อนที่ไว)     │
                      └───────────────────────────────────────────────┘
```

### หลักความสัมพันธ์ (เหตุผลออกแบบ)

1. **外功 ให้ atk — 內功 ให้ hp/def/內力**: เอนจินบวกสเตตัสจากทุกวิชารวมกัน
   (`recomputeStats`) ผู้ที่ฝึกแต่ 外功 จะตีแรงแต่ "กระดาษ" ส่วน 內功 ทำให้ยืนระยะได้
   → จึงต้องฝึก **คู่กัน** นี่คือความสัมพันธ์เชิงกลไกที่แท้จริง (synergy ผ่านการรวมสเตตัส)

2. **羅漢拳 = ราก**: หมัดอรหันต์เป็นวิชาแรก ราคาถูก ให้ทั้ง atk และ hp เล็กน้อย
   เป็นจุดเริ่มก่อนแตกเป็น 韋陀掌 (ฝ่ามือ) และ 達摩棍 (พลอง) — สื่อผ่านบทพูดพระคุยตี้/เสวียนเข่อ

3. **易筋經 = แกนกลางของสายใน**: ให้ 內力 (`maxMp_`) ซึ่งในโลกเกมคือเชื้อเพลิงของวิชาขั้นสูง
   เป็นบันไดสู่ 九陽神功 — สื่อผ่านปริศนาธรรมของหลวงจีนกวาดลาน

4. **九陽神功 = ขั้นสุดยอด แต่มีเงื่อนไข**: ราคาแพงสุด (✦15/rank, รวม 90 SP เต็มขั้น) +
   ออกแบบให้ต้องผ่านเควสต์เชนลับ (`shaolin_trial` → `shaolin_nine_yang`) ก่อน
   เควสต์มอบ SP ก้อนใหญ่ (รวม 80 SP) พอดีกับการไต่วิชานี้

5. **ปรัชญา "วิชาแรง+ปราณกำกับ"**: ปมเรื่องหัวหน้าครัวไฟ (ฝึก 外功 ล้วนจนเข้าสู่มาร)
   คือคำเตือนในเนื้อเรื่องว่าอย่าทิ้งสายใน — ผูกธีมกับกลไกข้อ 1 โดยตรง

### หมายเหตุข้อจำกัดเอนจินปัจจุบัน
ระบบ `canLearn` เช็กเพียง **rank ปัจจุบัน + แต้มวิชา (SP)** เท่านั้น ยัง**ไม่มี prerequisite แบบล็อกแข็ง**
(เช่น "ต้องฝึก 易筋經 ถึง rank 3 ก่อนจึงเรียน 九陽 ได้") ดังนั้นการ "ปลดล็อก" ในเอกสารนี้จึงเป็น
การ gate เชิงออกแบบผ่าน **ราคา SP + เควสต์เชน + เนื้อเรื่อง** หากต้องการล็อกแข็งในอนาคต
สามารถเพิ่มฟิลด์ `requires` ในไฟล์วิชาแล้วขยาย `canLearn` ให้ตรวจเงื่อนไขได้
